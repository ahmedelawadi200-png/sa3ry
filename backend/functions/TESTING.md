# دليل اختبار نظام تجميع الأسعار

## قبل الاختبار: تأكد من الخطوات دي
- [ ] رفعت ملفات الـ Frontend الجديدة على GitHub (js/admin.js, js/firebase.js, index.html, firestore.rules)
- [ ] نشرت `firestore.rules` الجديدة (Firebase Console → Firestore → Rules → الصق المحتوى → Publish)
- [ ] فعّلت خطة Blaze
- [ ] نشرت الـ Cloud Functions (`firebase deploy --only functions`)

---

## اختبار 1: أول تشغيلة (Manual Provider)

1. افتح لوحة الإدارة → **حالة المصادر**
2. دوس **"شغّل التحديث دلوقتي"**
3. **المتوقع:** توست أخضر "اتحدث X منتج"، وكارت جديد باسم `manual` بحالة "شغال" (أخضر)
4. كارت `amazon` المفروض يظهر بحالة "مش مفعّل" (رمادي) — ده صح ومتوقع، لسه معملتش الإعداد بتاعه

**لو حصل خطأ بدل كده:**
- افتح Firebase Console → Functions → Logs، ودور على أي سطر أحمر
- الأخطاء الشائعة: خطة Blaze مش مفعّلة، أو الـ Function اتنشرت غلط

## اختبار 2: التأكد إن المنتجات متأثرتش

1. روح للصفحة الرئيسية للتطبيق (مش لوحة الإدارة)
2. تأكد إن كل المنتجات القديمة لسه ظاهرة بنفس الأسعار والمحلات
3. **المهم:** أول تشغيلة بترجع تكتب نفس البيانات الموجودة (مفيش مصدر خارجي شغال لسه) — فمفروض ملحوظش أي تغيير في الموقع نفسه خالص

## اختبار 3: طابور المراجعة (Review Queue)

بما إن أول تشغيلة كانت بمصدر واحد بس (manual)، مش هيظهر أي حاجة في طابور المراجعة (محتاج مصدرين على الأقل عشان يقارن بينهم). ده هيبقى قابل للاختبار الحقيقي أول ما تضيف مصدر تاني (زي أمازون).

**اختبار جزئي دلوقتي:** افتح لوحة الإدارة → **طابور المراجعة** — المفروض تشوف رسالة "مفيش حاجة محتاجة مراجعة دلوقتي 🎉" (مش خطأ فاضي).

## اختبار 4: الجدولة التلقائية

1. Firebase Console → Functions → دوس على `scheduledPriceSync`
2. تأكد إن فيه **Trigger** من نوع Cloud Scheduler، وإن الجدول مكتوب `every 12 hours`
3. مش لازم تستنى 12 ساعة فعلياً عشان تتأكد — وجود الـ Trigger ده كافي دليل إنه هيشتغل

## اختبار 5: الأمان (تأكيد إن القواعد شغالة)

من متصفح **مش مسجل دخول بيه كأدمن**:
1. افتح Console (F12) وانت في الموقع
2. جرب تنفذ: `firebase.firestore().collection('providerRuns').get()`
3. **المتوقع:** خطأ `permission-denied` — لو رجعلك بيانات، القواعد مش متطبقة صح

---

## لما تضيف Amazon لاحقاً (اختبار مستقبلي)

1. حط الإعدادات (`firebase functions:config:set amazon...`) وانشر تاني
2. شغّل تحديث يدوي من لوحة الإدارة
3. المتوقع: كارت `amazon` يتحول لـ "شغال" (أخضر) في حالة المصادر
4. لو فيه منتجات متشابهة بين `manual` و`amazon` بنسبة 75-85%، هتظهر في طابور المراجعة تلقائي — دي أول مرة تقدر تختبر الميزة دي فعلياً بمصدرين حقيقيين

---

## نصيحة أخيرة
اختبر بالترتيب ده بالظبط (1 → 5) ومتقفزش خطوة، عشان لو حصلت مشكلة تعرف بالظبط في أنهي مرحلة حصلت.
